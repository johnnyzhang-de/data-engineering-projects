# Data Engineering Projects

This repository contains portfolio projects demonstrating modern data engineering pipelines.

## Project: GCS to BigQuery Data Pipeline

Architecture:

CSV file  
↓  
Google Cloud Storage  
↓  
Cloud Function (Python ingestion)  
↓  
BigQuery raw tables  
↓  
Airflow DAG transformation  
↓  
Analytics tables

## Technologies

- Python
- SQL
- Airflow
- Google Cloud Storage
- BigQuery
